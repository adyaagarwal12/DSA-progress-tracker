# LeetCode Tracker (C++)

A console-based LeetCode problem tracker built using C++ and OOP concepts.

## Features

- Add LeetCode problems
- View all stored problems
- Save problems to file
- Load previous records automatically

## Concepts Used

- OOP
- Classes and Objects
- Vectors
- File Handling
- Git and GitHub

## Project Structure

leetcode-tracker-cpp/

├── main.cpp
├── Problem.h
├── Tracker.h
├── Tracker.cpp
├── problems.txt
