# LeetCode Tracker (C++)

A console-based LeetCode problem tracker built using C++ and OOP concepts.

## Features

- Add LeetCode problems
- View saved problems
- Store problem title, topic, difficulty and solved status
- Save data to file
- Load data from file

## Technologies Used

- C++
- OOP
- File Handling
- STL Vector

## Project Structure

leetcode-tracker-cpp/

├── main.cpp  
├── Problem.h  
├── Tracker.h  
├── Tracker.cpp  
└── problems.txt
