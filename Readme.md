# LeetCode Tracker (C++)

A console-based LeetCode problem tracker built using C++ and OOP concepts.

## Features

- Add LeetCode problems
- View all stored problems
- Save problems to file
- Load previous records automatically

## Concepts Used

- Object Oriented Programming
- Classes and Objects
- Encapsulation
- Vectors
- File Handling
- Multi-file project structure
- Git and GitHub

## Project Structure

├── main.cpp
├── Problem.h
