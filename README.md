# LeetCode Tracker (C++)

A console-based LeetCode problem tracker built using **C++**, **OOP**, and **file handling** concepts.

## Features

- Add LeetCode problems
- View saved problems
- Search problems by topic
- Delete problems
- Save data to file
- Load saved data automatically on startup
- Statistics dashboard

## Concepts Used

- Object Oriented Programming (OOP)
- Classes and Objects
- Encapsulation
- STL Vector
- File Handling (`ifstream`, `ofstream`)
- Loops and Conditions
- Git and GitHub
